Introduction

This guide is designed to help you navigate the resources in this API and showing you how to make different types of queries. Please note that while the GET methods can be accessed directly from this page, the POST, PUT, and DELETE methods require the use of Swagger.

  • Base url: https://localhost:7003
  • Swagger url: https://localhost:7003/swagger/index.html

Endpoints

  1. Get all artists
  2. Get artist by id
  3. Get artist by name
  4. Post an artist
  5. Update an artist
  6. Delete an artist

Get all artists

You can access all artists by using the /artist endpoint.

Example

GET: https://localhost:7003/Artist 

                    
[
    {
        id: 1,
        artistName: "Astrid S",
        genre: "Pop",
        image: "astrids.png"
    },
    {
        id: 2,
        artistName: "Dagny",
        genre: "pop",
        image: "dagny-png"
    },
    {
        id: 3,
        artistName: "Sigrid",
        genre: "Pop",
        image: "sigrid.png"
    }
]

Get an artist by id

You can access a single artist by using the artists id as a parameter.

Example

GET: https://localhost:7003/Artist/2 

                    
{
    id: 2,
    artistName: "Dagny",
    genre: "pop",
    image: "dagny.png"
}

Get an artist by name

You can access an artist (or several if the name has more than one matches) by searching for his/her name.

Example

GET: https://localhost:7003/Artist/GetArtistByName?artistName=arif 

                    
{
    id: 5,
    artistName: "Arif",
    genre: "RnB",
    image: "arif.png"
}

Post an artist

You can post an artist by adding all parameters: artistName, genre and image. It is possible to post without image, but note that it's not recommended. Id is auto-incremented by default, and can be left at 0.

Example

POST: https://localhost:7003/Artist 

                    
{
    id: int,
    artistName: "string",
    genre: "string",
    image: "string"
}

Update an artist

You can update an artist by altering the parameters you want to change. You won't be able to alter the id as it is auto-incremented in the database, but artist name, genre and image can be changed.

Example

PUT: https://localhost:7003/Artist 

                    
{
    id: int,
    artistName: "string",
    genre: "string",
    image: "string"
}

Delete an artist

You can delete an artist by providing the id of the artist you want to delete from the database.

Example

DELETE: https://localhost:7003/Artist 

                    
{
    id: int
}
